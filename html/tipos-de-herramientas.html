Tipos de herramientas

Material

Podemos clasificarlas de la siguiente manera:

Herramientas de montaje: Sirven para ensamblar o desensamblar muebles, partes o aparatos, como son por ejemplo los destornilladores o las llaves.
Herramientas de sujeción: Sirven para sujetar piezas o materiales y ejercer una mayor fuerza de manera segura. Por ejemplo las pinzas y los alicates.
Herramientas de golpe: Sirven para aplicar una fuerza sobre una superficie, potenciando el impacto que podría tener la mano humana. Son ejemplo de ello el martillo y el pico.
Herramientas de corte: Sirven para separar o cortar materiales, como es el caso de las tijeras o cuchillos.
Herramientas de unión, que sirven para juntar piezas o materiales difíciles de acoplar, como lo hacen por ejemplo los sopletes o las pistolas de silicona.
Herramientas de medición: Sirven para medir superficies o comparar magnitudes con otras. Es el caso de los compases, las cintas métricas y las escuadras.
